Dify 团队在最新的 0.6.11 版本中与 Firecrawl 达成合作，为知识库引入了同步 Web 站点数据的全新功能。这一更新不仅丰富了知识库的数据来源，还为用户提供了更加便捷、高效的信息获取方式，大大提升了用户体验。此外，本次版本更新还支持在 Workflow 中添加注释，从而增强了团队协作的效率和便利性。

Firecrawl：连接 Web 数据与 RAG 应用

Firecrawl 是一款强大的网页爬取和转换工具，它可以将任何网站的内容提取并转换为干净、结构化的 Markdown 格式。通过 Firecrawl 处理后的网站数据，非常适合作为 LLM 的上下文信息使用。你可以利用这些高质量的结构化数据，在 Dify 平台上便捷地构建 RAG 应用。

API 和功能配置

Dify 的知识库页面新增了"同步自 Web 站点"的选项，只需从 Firecrawl 获取 API 密钥即可完成配置。值得一提的是，Firecrawl 的云服务 API 还提供了 500 个免费页面的抓取额度，为用户提供了充足的试用机会。
