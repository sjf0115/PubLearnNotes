## 1. 功能

### 1.1 概览

在用户管理界面，可以直接查看已加入画像平台的用户数量、管理员数量、成员数量、以及无权限用户数量，如下图：

![](https://lf6-volc-editor.volccdn.com/obj/volcfe/sop-public/upload_e529b3f33c12fc9ca4a703c40e7b24ef.png)

### 1.2 用户列表

同时还可看到所有进入画像平台的所有用户列表。用户列表中的信息包括了用户的姓名、角色、状态以及操作，如下图：

![](https://lf6-volc-editor.volccdn.com/obj/volcfe/sop-public/upload_1a6d5e3e6dff4bbd11dbcd9ba5109a9c.png)

#### 1.2.1 编辑&删除成员角色信息

拥有管理员权限的用户，可以在操作栏点击“编辑”，来进行其他成员角色信息的添加或修改，如下图。

![](https://lf6-volc-editor.volccdn.com/obj/volcfe/sop-public/upload_7be9eed37a5a2440dcac28843bf42219.png)

### 1.2 邀请用户

拥有邀请权限的成员，可点击用户列表中的“邀请用户”按钮，填写新用户的“邮箱”，并为其添加完相应的角色信息后，点击“确认”完成邀请流程。如需同时添加多个用户，请点击窗口下方“+添加新用户”按钮。

![](https://lf3-volc-editor.volccdn.com/obj/volcfe/sop-public/upload_9682bd7e9f084cf1466db16d13cebc83.png)

批量邀请用户
- 支持同时邀请多个用户，每个用户需填写邮箱和选择角色
- 点击「+添加新用户」可添加更多用户
- 每个用户卡片可独立删除（至少保留一个）
点击「邀请用户」按钮
填写用户邮箱（必填）
选择角色（可多选，必填）
点击「+添加新用户」添加更多用户
点击「确定」批量邀请所有有效用户

## 1. 数据库设计

| 字段 | 参数 | 说明 |
| :------------- | :------------- | :------------- |
| id | 自增ID  |  |
| status | 是否有效  | 启用-1,停用-2 |
| user_id | 用户ID  |  |
| user_name | 用户名称  |  |
| source_type | 创建方式  | 系统内置:1,自定义:2 |
| creator | 创建者  |  |
| modifier | 修改者  |  |
| gmt_create | 创建时间  |  |
| gmt_modified | 修改时间  |  |


```sql
DROP Table `profile_meta_user`;
CREATE TABLE `profile_meta_user` (
  `id` bigint unsigned NOT NULL AUTO_INCREMENT COMMENT '自增ID',
  `status` int NOT NULL DEFAULT '1' COMMENT '状态:1-未激活,1-激活,2-停用',
  `user_id` varchar(40) NOT NULL COMMENT '用户ID',
  `user_name` varchar(100) NOT NULL COMMENT '用户名称',
  `email` varchar(100) NOT NULL COMMENT '邮箱',
  `password` varchar(100) NOT NULL COMMENT '密码',
  `source_type` int NOT NULL DEFAULT '1' COMMENT '创建方式: 1-系统内置,2-自定义',
  `creator` varchar(100) NOT NULL COMMENT '创建者',
  `modifier` varchar(100) NOT NULL COMMENT '修改者',
  `gmt_create` datetime NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  `gmt_modified` datetime NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '修改时间',
  PRIMARY KEY (`id`),
  UNIQUE KEY `user_id` (`user_id`)
) ENGINE=InnoDB AUTO_INCREMENT=1 DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_0900_ai_ci COMMENT='画像-用户'
```

初始化内置管理员：
```sql
INSERT INTO `profile_meta_user` (`status`, `user_id`, `user_name`, `email`, `password`, `source_type`, `creator`, `modifier`)
VALUES (2, '100000', 'admin', '', 'admin', 1, '100000', '100000');
```





## 2. Mapper

```java
@Mapper
public interface UserMapper {
    // 查询
    User selectByUserId(String UserId); // 根据ID查询
    List<User> selectByUserName(String UserName); // 根据名字查询
    List<User> selectByParams(User User); //根据参数查询
    List<User> selectByKeyword(String keyword); // 模糊查询
    // 插入
    int insert(User User); // 插入全部
    int insertSelective(User User); // 选择性插入
    // 删除
    int deleteByUserId(String UserId);
    // 更新
    int updateByUserId(User User);
    int updateByUserIdSelective(User User);
}
```
